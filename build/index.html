MY WORK

SongQuest

A full-stack web application that revolutionizes the way users find music rights information for music licensing, with the aim of streamlining the process and providing a centralized platform. Developed within a Django backend framework using Selenium, Beautiful Soup, and Spotify's API to seamlessly gather and present accurate music rights data. This information is conveniently accessible to users through a user-friendly frontend search interface built with React.js, React Redux, and MaterialUI, ensuring an intuitive experience.

QuotaPath

A compensation management web application that provides sales teams with accurate and up to date deal and compensation data to inform and motivate teams through transparent reporting, and forecasting of deals, revenue, and commissions. Enabled input and calculation of deal data for clear reporting display and exports that users could leverage to track individual and team sales performance in order to accurately manage compensation and motivate team members in attaining goals and milestones.

BarkSquad

A front-end web application that displays profile information about dogs. Developed using React.js and Redux. Reducer functions include adding a dog, updating a dog's information, deleting a dog, filtering dog's by size, searching for a dog by it's name, and sorting dog's by name. Also allows the ability to view individual dog profiles by clicking on the specified dog's profile card.
